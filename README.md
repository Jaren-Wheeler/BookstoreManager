This project is a Bookstore Manager, allowing bookstores to manage their inventory, track customers, and have records of sales. The project is designed in MVVM architecture, a three-layered structure composed of View, ViewModels, and Models.

Project Specifications:
- C#/.NET 4.8
- Windows Presentation Foundation Framework
- Integrated SQLite database

Setup Instructions:
1. Clone the repository into your local machine
2. Ensure Visual Studio is installed
3. If the proper packages aren't already available, install via NuGet Package Manager (Tools -> NuGet Package Manager -> Manage NuGet Packages for Solution). The required dependencies are:
   - Microsoft.Data.Sqlite.Core
   - SQLitePCLRaw.lib.e_sqlite3
